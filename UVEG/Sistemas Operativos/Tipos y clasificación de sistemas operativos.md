Los [[OS]] no sólo se pueden clasificar por su arquitectura, sino que también se pueden clasificar en: 
- [[Mainframe]] 
- [[De servidor]] 
- [[Multiprocesador]] 
- [[Ordenador personal ]]
- [[De tiempo real]] 
- [[Empotrados]] 
- [[De tarjeta inteligente]]

[[Mainframe]]
- Principalmente para grandes orgs para gestionar cantidades grandes de información
- Son computadoras con un alto nivel de procesamiento
- A menudo usan máquinas virtuales con [[SO]] especializados en ofrecer alto desempeño para procesamiento de transacciones
- Ejemplos:
	- z/OS
	- IBM i as400
	- ISPF


[[De servidor]]
- Optimizados para permitir operar en base a [[Cliente-servidor]] 
- Se usan principalmente en páginas web, servidores de transferencia de archivos, aplicaciones web, etc.
- Ejemplos:
	- Windows Server
	- CentOS
	- Ubuntu Server
	- Red Hat

[[Multiprocesador]]
- AKA ordenadores paralelos o multicomputadoras
- SO generalmente de servidor, con algunas variaciones para usar varias CPU
- Son de uso muy difundido
- Por ejemplo:
	- Windows
	- MacOS
	- Linux

[[Ordenador personal]]
- Son los más comunes
- Con interfaz amigable
- Se usan para la conectividad a internet, procesamiento de texto y hojas de cálculo
- Ejemplos: 
	- Windows 10
	- Ubuntu
	- macOS Mojave

[[De tiempo real]]
- El tiempo real es su parámetro clave
- Las acciones se deben realizar en un momento exacto, sino se daña la producción
- Se usan en maquilas, uso militar, etc
- Ejemplos:
	- MaRTE OS
	- LynxOS
	- FreeRTOS
	- RT-11
	- QNX

[[Empotrados]]
- AKA embebidos o integrados
- No se les considera computadoras y no permiten instalar software por un usuario
- Ejemplos:
	- Hornos de microondas
	- Lavadoras
	- Autos

[[De tarjeta inteligente]]
- Son los más pequeños
- Están presentes en tarjetas bancarias o más pequeñas
- Funciones muy limitadas
- Ejemplos:
	- TDC
	- TDD
