
This is the domain controller.

If there are network issues (no internet connection) disable and enable the net adapter:
```
Disable-NetAdapter -Name "Ethernet0" -Confirm:$false; Enable-NetAdapter -Name "Ethernet0"
```


# Create domain user
```
New-ADUser -Name "name of the user" -AccountPassword (ConvertTo-SecureString "Password123!" -AsPlainText -Force) -Enabled $true -SamAccountName newuser
```

# Organizational Units (OU)
Create a OU
```
New-ADOrganizationalUnit -Name "Workstations" -Path "DC=sapito,DC=lab"
```

Move a PC into a OU
```
Move-ADObject -Identity "CN=PC01,CN=Computers,DC=sapito,DC=lab" -TargetPath "OU=Workstations,DC=sapito,DC=lab"
```

Automatically redirect computers to the Workstations OU
```
RedirCmp "OU=Workstations,DC=sapito,DC=lab"
```
