
[[Editores de código]]
- Herramientas que permiten simplificar el coding.
- Permiten la edición de lenguajes de programación.
- Syntax verification in real time.
- Facilita la tarea del desarrollo.
- Coloreado de syntax.
- Autocompletado.
- Indentación. 

[[IDE]]
- Ayuda en el diseño y desarrollo.
	- Editor de código, compilador, intérprete, depurador, cliente.

[[Compilador]]
- Traducción desde lenguaje de alto nivel a lenguaje máquina.

[[Intérprete]]
- Traducen cada una de las sentences para después ejecutarlas.
- Primero pasan a máquina, y de ahí ejecutan la sentencia. (Cada una de las instrucciones).
- Son más lentos que el código máquina.
- Pero es más sencilla y más sencillo de detectar errores.

[[Consola y línea de comandos]]
- El CMD, bash, shell, zsh, etc.
- Directamente al [[SO]].
