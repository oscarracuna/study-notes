Hardware:
- Cualquier clase de dispositivo físico, ya sea eléctrico, electrónico o mecánico, asociado con la computación.
Software:
- Programa de cómputo o conjunto de instrucciones que se ejecutan en una computadora para realizar una tarea específica.
Telecomunicaciones:
- Dispositivos, reglas y protocolos que procesan señales con el objetivo de ser transmitidas.
Información:
- Conjunto de datos agrupados que pueden ser procesados por un sistema computacional.


Metodologías en desarrollo de software
- Modelo en cascada
- Modelo espiral
- Modelo orientado a objetos UML
- Desarrollo ágil SCRUM
- Extreme programming
