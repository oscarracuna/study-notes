
- Archivo o conjunto de archivos que utilizan instrucciones o tareas que se encuentran codificadas en lenguaje de programación ejecutado or una computadora.

**Clasificación del [[Software]]**
- [[Software]] de aplicación:
	- Tiene un propósito específico.
- [[Software]] del sistema:
	- Auxilia en determinadas tareas y funciones.

**Estructura del [[software**]]
- Cabecera o [[header]]:
	- Contiene el nombre del programa y metadatos asociados.
- Cuerpo del programa o [[body]]:
	- Declaraciones
		- Constantes, tipos de datos y variables.
	- Instrucciones
		- Instrucciones a ejecutar para obtener resultados.

**Instrucciones / acciones básicas**
- Inicio
- Asignación
	- Dar valor a una variable.
- Lectura
	- Toma input y almacena en memoria.
- Escritura
	- Envía datos a un dispositivo de output.
- Bifurcación
	- Permite controlar que se ejecuten on no otras instrucciones.
