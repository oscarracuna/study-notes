
## ¿Qué es una base de datos?
Conjunto de información organizada de tal manera que sea eficiente.
Por ejemplo:
- Diccionarios
- Directorio telefónico
- Calendario

## ¿Qué es un [[dato]]?
Palabras y/o números que se integran entre sí para estructurar información para posteriormente consultar

## Objetivo de las [[bases de datos]]
Generar un mecanismo de almacenamiento de datos que permitan que la información sea recopilada, organizada, ordenada y almacenada, procesada y consultada de manera ágil.
