
Most commands have an available short description which can be viewed using the `--help` or `-h` option.
For example:
```bash
man --help
```

# The help Command

When running a [[bash]] shell, some popular commands (`echo`, `cd`) run especially built-in bash versions of the commands instead of the binaries in `/bin` or `/usr/bin` for efficiency purposes.

So, to view a synopsis of these built-in commands you can type `help`.
