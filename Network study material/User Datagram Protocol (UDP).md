Communications protocol for time-sensitive applications, like **gaming, playing videos or *Domain Name System (DNS)***. 
	-Speedier communication since it doesn't spend time forming a firm connection with the destination.
	- UDP can cause data packets to get lost.
	- It's relatively easy to suffer a DDoS attack.

**How does it work?**
A target computer is identified and the data packets -datagrams- are sent to it.
