**¿Qué es un arreglo?**
Los arreglos son una lista de datos del mismo tipo, los cuales nos sirven para agrupar variables. Los arreglos nos permiten almacenar tantos datos como queramos en una variable.

**Declaración y creación**
   **Definición**

```java
int [] nombre1;
int nombre2[];
```

   **Creación**
```java
nombre_1 = new int[5];
nombre_2 = new int[5];
```

**Acceso a los elementos**
La posición de los arreglos siempre empieza en 0 y les llamamos índices.

[[Arreglos unidimensionales]]
Son objetos donde podemos almacenar varios valores, todos apuntando a la misma variable, diferenciados por su posición, mejor conocida como índice.

Para hacer el recorrido de los elementos de un arreglo usamos las funciones iterativas, for, while, do while, etc.
