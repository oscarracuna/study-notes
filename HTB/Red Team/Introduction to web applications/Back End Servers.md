
The hardware and [[OS]] on the back end that hosts all of the applications necessary to run the web app. It would fit in the [[Data access]] layer.


**Software**
It contains 3 other components:
- Web server
- [[database]]
- Development [[Frameworks]]

Other components may include:
- [[Hypervisors]]
- [[Containers]]
- [[WAF]] 

There are many stacks, for example:

- LAMP
	- Linux, Apache, MySQL and PHP
- WAMP
	- Windows, Apache, MySQL and PHP
- WINS
	- Windows, IIS, .NET and SQL Server
- MAMP
	- macOS, Apache, MySQL and PHP
- XAMPP
	- Cross-platform, Apache, MySQL and PHP/PERL
